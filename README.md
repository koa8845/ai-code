# ระบบคำนวณค่าจัดส่งสินค้า (Shipping Calculator)

## รายละเอียดโปรแกรม

**ShippingCalculator.java** เป็นโปรแกรมที่ใช้ในการคำนวณค่าจัดส่งสินค้าตามเงื่อนไขต่าง ๆ ได้แก่ สถานที่ปลายทาง น้ำหนักของพัสดุ ประเภทการจัดส่ง และสถานะสมาชิก โดยโปรแกรมถูกออกแบบให้มีความยืดหยุ่นและสามารถนำไปประยุกต์ใช้งานได้ง่าย

## โครงสร้างคลาส

### ค่าคงที่ (Constants)
- `BANGKOK_AREA`: ค่าคงที่สำหรับระบุพื้นที่กรุงเทพฯ และปริมณฑล
- `OTHER_PROVINCE`: ค่าคงที่สำหรับระบุพื้นที่ต่างจังหวัด
- `STANDARD`: ค่าคงที่สำหรับระบุบริการมาตรฐาน
- `EXPRESS`: ค่าคงที่สำหรับระบุบริการด่วน
- `REGULAR`: ค่าคงที่สำหรับระบุลูกค้าทั่วไป
- `VIP`: ค่าคงที่สำหรับระบุลูกค้าระดับ VIP

## ฟังก์ชันหลัก

### 1. calculateShippingCost()
```java
public double calculateShippingCost(String location, double weight, String serviceType, String membership)
```
**หน้าที่**: คำนวณค่าจัดส่งตามพารามิเตอร์ที่กำหนด

**ขั้นตอนการทำงาน**:
1. ตรวจสอบความถูกต้องของข้อมูลนำเข้า (input validation)
2. คำนวณต้นทุนพื้นฐานจากตำแหน่งปลายทางและน้ำหนัก
3. เพิ่มค่าบริการด่วน 30 บาท หากเลือกบริการด่วน
4. หักส่วนลด 20% หากเป็นสมาชิก VIP
5. ปัดเศษผลลัพธ์เป็นทศนิยม 2 ตำแหน่งแล้วคืนค่า

### 2. calculateBaseCost()
```java
private double calculateBaseCost(String location, double weight)
```
**หน้าที่**: คำนวณต้นทุนพื้นฐานตามสถานที่และน้ำหนัก

**ขั้นตอนการทำงาน**:
- เรียกใช้ฟังก์ชัน `getBangkokAreaCost()` หากเป็นพื้นที่กรุงเทพฯ และปริมณฑล
- เรียกใช้ฟังก์ชัน `getOtherProvinceCost()` หากเป็นต่างจังหวัด

### 3. getBangkokAreaCost()
```java
private double getBangkokAreaCost(double weight)
```
**หน้าที่**: คำนวณค่าจัดส่งสำหรับพื้นที่กรุงเทพฯ และปริมณฑลตามเกณฑ์:
- น้ำหนัก 0–1 กิโลกรัม = 40 บาท
- น้ำหนักมากกว่า 1–3 กิโลกรัม = 60 บาท
- น้ำหนักมากกว่า 3–5 กิโลกรัม = 80 บาท
- น้ำหนักมากกว่า 5 กิโลกรัม = 100 บาท

### 4. getOtherProvinceCost()
```java
private double getOtherProvinceCost(double weight)
```
**หน้าที่**: คำนวณค่าจัดส่งสำหรับต่างจังหวัดตามเกณฑ์:
- น้ำหนัก 0–1 กิโลกรัม = 60 บาท
- น้ำหนักมากกว่า 1–3 กิโลกรัม = 90 บาท
- น้ำหนักมากกว่า 3–5 กิโลกรัม = 120 บาท
- น้ำหนักมากกว่า 5 กิโลกรัม = 150 บาท

### 5. displayCostBreakdown()
```java
public void displayCostBreakdown(String location, double weight, String serviceType, String membership)
```
**หน้าที่**: แสดงรายละเอียดการคำนวณค่าจัดส่งแบบละเอียด รวมถึง:
- สถานที่ปลายทาง
- น้ำหนักของพัสดุ
- ประเภทบริการ
- สถานะสมาชิก
- ต้นทุนพื้นฐาน
- ค่าบริการด่วน (ถ้ามี)
- ยอดรวมก่อนหักส่วนลด
- ส่วนลด VIP (ถ้ามี)
- ราคารวมสุดท้าย

## การตรวจสอบความถูกต้องของข้อมูลนำเข้า

โปรแกรมมีการตรวจสอบข้อมูลนำเข้าอย่างครบถ้วนผ่านฟังก์ชัน:
- `isValidLocation()`: ตรวจสอบว่าสถานที่อยู่ในรายการที่รองรับ
- `isValidServiceType()`: ตรวจสอบว่าประเภทบริการถูกต้อง
- `isValidMembership()`: ตรวจสอบว่าสถานะสมาชิกถูกต้อง

## คุณสมบัติเด่นของโปรแกรม

1. **การจัดการข้อผิดพลาด**: มีการตรวจสอบข้อมูลนำเข้าและแจ้งข้อผิดพลาดเมื่อพบข้อมูลไม่ถูกต้อง
2. **การยืดหยุ่น**: รองรับการเปลี่ยนแปลงเงื่อนไขการคิดค่าบริการโดยไม่ต้องแก้ไขโค้ดหลัก
3. **ความสามารถในการอ่าน**: โค้ดถูกจัดโครงสร้างให้อ่านเข้าใจง่ายด้วยชื่อฟังก์ชันและตัวแปรที่ชัดเจน
4. **การแสดงผลแบบละเอียด**: มีฟังก์ชันแสดงรายละเอียดการคำนวณเพื่อให้ผู้ใช้เข้าใจการคิดคำนวณ

## การใช้งาน

โปรแกรมสามารถนำไปใช้ในระบบจัดส่งสินค้าโดยเรียกใช้ฟังก์ชัน `calculateShippingCost()` พร้อมกับส่งค่า:
- สถานที่ปลายทาง
- น้ำหนักของพัสดุ
- ประเภทการจัดส่ง
- สถานะสมาชิก

โปรแกรมจะคืนค่าเป็นจำนวนเงินค่าจัดส่งตามเงื่อนไขที่กำหนด
